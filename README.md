## Go Ethereum Tools

A collection of tools for go ethereum (geth).

## Usage Examples

Get balances of all accounts as well as a total balance
```
geth --preload getBalances.js attach

// Balances in wei
getBalances("wei");

// Balances in gwei
getBalances("gwei");

// Balances in Ether
getBalances("ether");
```
